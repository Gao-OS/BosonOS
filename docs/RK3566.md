# RK3566 Direction

Initial RK3566 support is modeled, not fully implemented.

SoC:

- `rockchip-rk3566`
- `cortex-a55`
- Linux `Image`
- U-Boot
- rkbin firmware family
- `ttyS2,1500000`

Initial boards:

- Powkiddy RGB30
- Powkiddy X55
- Anbernic RG353V

ROCKNIX resources may be imported under `third_party/rocknix` for kernel
config, patches, DTS files, board lists, and boot image conventions. BosonOS
must still build kernels and images through its own Nix abstractions.
